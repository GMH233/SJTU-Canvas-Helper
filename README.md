## Main Features
+ [x] 文件下载
+ [x] 人员名单导出
+ [x] 查看作业
+ [x] 学生提交作业批量下载
+ [x] 视频下载

### 预览

类似 macOS quick look，按下空格打开预览，再次按下空格关闭预览。
